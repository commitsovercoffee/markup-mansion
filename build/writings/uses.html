What I Use

Get the inside scoop on the tools that make my life easier.

Before we begin with the list, please note that the items listed below are ones that I personally use. And thus it is only a matter of personal preference.

People on the internet seem to make a big deal about what os or editor you use but at the end of the day, it’s about what you create, not the tools you use. Like the saying goes, whatever floats your boat. ⛵

Software

I have a highly customized arch linux build that I have been daily driving for over 5 years. You can find the repo with my dot files and other details here, but below is the gist :

  • OS : arch linux.
  • Desktop Environment : dynamic window manager.
  • Terminal : simple terminal with fish shell & starship prompt.
  • Text Editor : neovim (or neovide).
  • Theme : catpuccin.
  • Font : fira code.
